# @word-kit/core

OOXML-compliant (ECMA-376) `.docx` generation for browsers and Node.js.

See the [repository README](../../README.md) for the project overview.

## Status

Pre-1.0. Public API is unstable until v1.

## Install

```bash
npm install @word-kit/core
# or
pnpm add @word-kit/core
```
